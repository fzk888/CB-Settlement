# Phase 1 平台字段对照表

> 版本: Phase 1 多平台扩展  
> 更新日期: 2026-02-07

本文档定义各平台账单字段映射及**收入计算公式**。

---

## 一、Amazon

### 文件格式
- 格式: CSV
- 多语言支持: 英/德/法/日

### 收入计算公式

```
平台净结算 = Σ (所有非Transfer/Payout交易的 Total 字段)
```

### 各字段定义

| 报表字段 | 计算公式 |
|---------|----------|
| **销售收入** | `Product Sales` + `Postage Credits` + `Shipping Credits` + `Gift Wrap Credits` |
| **平台费用** | `Selling Fees` + `FBA Fees` + `Other Transaction Fees` |
| **促销折扣** | `Promotional Rebates` |
| **平台净结算** | `Σ Total` (排除 Transfer/Payout) |

### 排除规则
- `type` 字段含 `Transfer` 或 `Payout`

---

## 二、Temu

### 文件格式
- 格式: Excel (.xlsx)
- 文件名特征: `FundDetail-xxx.xlsx`
- 结构: 多 Sheet，每个 Sheet 代表一种收支类型

### 收入计算公式

```
平台净结算 = 结算-交易收入
           + 结算-运费收入
           - |结算-售后退款|
           - |结算-运费退款|
           - |支出-履约违规|
           - |支出-技术服务费|
```

### Sheet 与金额方向

| Sheet 名称 | 金额方向 | 说明 |
|------------|---------|------|
| 结算-交易收入 | **正数** | 商品销售收入 |
| 结算-运费收入 | **正数** | 运费收入 |
| 结算-售后退款 | **负数** | 退款扣减 |
| 结算-运费退款 | **负数** | 运费退款 |
| 支出-履约违规 | **负数** | 违规罚款 |
| 支出-技术服务费 | **负数** | 技术服务费 |

### 计算示例 (All F Home 2025-07)

| Sheet | 金额 (USD) |
|-------|-----------|
| 结算-交易收入 | +3,173.41 |
| 结算-运费收入 | +922.28 |
| 结算-售后退款 | -203.18 |
| 结算-运费退款 | -42.89 |
| 支出-履约违规 | -24.00 |
| **平台净结算** | **3,825.62** |

### 排除规则
- **无**：Temu 账单不含提现记录，全部参与计算

---

## 三、SHEIN

### 文件格式
- 格式: Excel (.xlsx)
- 文件名特征: `已完成账单-账单商品维度-供货价-xxx.xlsx`
- 结构: 单 Sheet，首行为汇总数据

### 收入计算公式

```
平台净结算 = Σ 应收金额
```

### 关键字段

| 列名 | 说明 |
|------|------|
| 商品价格汇总 | 客户支付的商品价格 |
| 商品供货价汇总 | 供货商供货价 |
| 活动优惠金额 | 活动折扣（负数） |
| 活动提报金额 | 活动费用（负数） |
| 服务费 | 平台服务费（负数） |
| 履约服务费 | 履约费用（负数） |
| **应收金额** | **最终结算金额** |

> 注：SHEIN 账单每行已计算好 `应收金额`，直接求和即可

### 计算示例 (天基希音UK 2025-07)

| 字段 | 金额 (GBP) |
|------|-----------|
| 记录数 | 1,334 条 |
| 应收金额合计 | 15,135.61 |
| **平台净结算** | **15,135.61** |

### 排除规则
- **无**：SHEIN 账单不含提现记录，全部参与计算

---

## 四、托管店铺

### 文件格式
- 格式: Excel (.xlsx)
- 文件名特征: `收支明细_xxx.xlsx`
- 结构: 每行一笔费用记录

### 收入计算公式

```
平台净结算 = Σ 金额(CNY)，其中 费用项 ≠ '提现'
```

### 费用项分类

| 费用项 | 金额方向 | 说明 |
|--------|---------|------|
| 供货款 | **正数** | 商品销售收入 |
| 供货款退款 | **负数** | 退货退款 |
| 海外发货处罚 | **负数** | 违规罚款 |
| 退平台仓质检费 | **负数** | 质检费 |
| 退平台仓运费 | **负数** | 仓储运费 |
| 提现 | -- | **排除不计** |

### 计算示例 (天基托管 2025-07)

| 费用项 | 金额 (CNY) |
|--------|-----------|
| 供货款 | +14,524.68 |
| 供货款退款 | -367.97 |
| 海外发货处罚 | -247.06 |
| 退平台仓质检费 | -96.39 |
| 退平台仓运费 | -197.43 |
| **平台净结算** | **13,615.83** |

### 排除规则
- `费用项` = `提现`

---

## 五、速卖通

### 文件格式
- 格式: Excel (.xlsx)
- 文件名特征: `收支流水xxx.xlsx`
- 结构: 每行一笔收支记录

### 收入计算公式

```
平台净结算 = Σ 变动金额，其中 收支类型 不含 '提现'
```

> 注：`变动金额` 格式为 `CN￥ 297.00`，需去除前缀解析

### 收支类型分类

| 收支类型 | 费用项 | 金额方向 |
|----------|--------|---------|
| 收入>放款 | 供货款放款 | **正数** |
| 支出>退款 | * | **负数** |
| 支出>平台处罚 | * | **负数** |
| 支出>物流费用 | * | **负数** |
| 支出>提现 | * | **排除不计** |

### 计算示例 (速卖通 2025-09)

| 收支类型 | 金额 (CNY) |
|----------|-----------|
| 收入>放款 | +8,253.18 |
| 支出>放款后退款 | -133.06 |
| 支出>平台处罚 | -129.81 |
| 支出>物流费用 | -101.50 |
| **平台净结算** | **7,888.81** |

### 排除规则
- `收支类型` 含 `提现`

---

## 六、排除规则汇总

| 平台 | 排除条件 | 说明 |
|------|----------|------|
| Amazon | type 含 `Transfer` / `Payout` | 银行提现 |
| Temu | 无 | 账单不含提现记录 |
| SHEIN | 无 | 账单不含提现记录 |
| 托管店铺 | 费用项 = `提现` | 银行提现 |
| 速卖通 | 收支类型 含 `提现` | 银行提现 |

---

## 七、计算公式总览

| 平台 | 核心公式 | 币种 |
|------|---------|------|
| Amazon | `Σ Total` (排除 Transfer/Payout) | 多币种 |
| Temu | `结算交易收入 + 结算运费收入 - 退款 - 支出` | USD |
| SHEIN | `Σ 应收金额` | 按站点 |
| 托管店铺 | `Σ 金额(CNY)` (排除提现) | CNY |
| 速卖通 | `Σ 变动金额` (排除提现) | CNY |
